# Quick reference

| Action | How |
|---|---|
| Add a doc to this namespace | Sidebar → under `demos/` → **+ entry** |
| Edit this namespace's llms.txt | Sidebar → `demos/` → `llms.txt` |
| Point an agent at this namespace | `GET /demos/llms.txt` |
| Point an agent at everything | `GET /llms.txt?merge=true` |
| Delete this namespace | Sidebar → `×` next to `demos/` |
